Titanic Survival Analysis System
鐵達尼號生存數據多維度分析系統

---


核心技術
資料處理**：使用 `Pandas` 進行數據清洗與結構化篩選。
資料視覺化**：整合 `Matplotlib` 與 `Seaborn` 繪製圖表。
開發邏輯**：採用模組化函式（Functional Programming）設計。

主要功能
1. 精確名單篩選**：自動區分成年男女生存/死亡名單。
2. 視覺化洞察**：包含結構分析、社會階層關聯及年齡分佈密度。

---



Tech Stack
* **Data Processing**: `Pandas` for cleaning and structured filtering.
* **Visualization**: `Matplotlib` and `Seaborn` for data plotting.
* **Programming Logic**: Modular functional programming approach.

Key Features
1. **Precise Filtering**: Categorizes detailed survivor and casualty lists.
2. **Visual Insights**: Provides class correlation and age distribution density.

---

Data Insights / 數據結論
* **Gender**: 女性存活率顯著高於男性 (Female survival significantly higher).
* **Class**: 三等艙死亡風險最高 (3rd class faced highest risk).
